# [[11049번]](https://www.acmicpc.net/problem/11049) 행렬 곱셈 순서

## 해결 방법
바텀업 DP를 이용한다.
길이 2부터 n까지 각 부분 구간별 최소 연산 횟수를 구한다.
a부터 b까지의 최소 연산 횟수는 dp[a][a + k] + dp[a + k + 1][b] + r[a] * c[a + k] * c[b](0 <= k < b - a)이다.
---
### 1. 각 길이별 최소값을 구한다.

- 3중 for문을 이용한다.
- 첫 번째는 부분 구간의 길이 (b - a)
- 두 번째는 시작 인덱스 (b)
- 세 번째는 k

### 2. 정답 출력

- 1번이 끝나면 dp[0][n-1]에 정답이 저장되어 있다.
